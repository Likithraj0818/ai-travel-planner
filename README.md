# ✈️ AI-Powered Travel Planner

> Plan your perfect trip in seconds — just enter your destination, budget, and dates. AI agents handle the rest.

Built with **CrewAI**, **Google Gemini 1.5 Flash**, **LangChain**, and **Streamlit** — with real-time web search via **SerperDev**.

---

## 📌 What This Project Does

A multi-agent AI system that generates fully personalized travel itineraries. You provide:
- Destination
- Trip duration
- Budget
- Preferences (e.g. historical sites, food, adventure)

The AI returns a complete travel plan with attractions, hotel suggestions, weather info, and daily schedule.

**Example:**
```
Input:  "Plan a 3-day trip to London with a $1500 budget. I prefer historical sites."
Output: Complete itinerary with places to visit, hotel near public transport,
        weather forecast, budget breakdown, and daily schedule.
```

---

## 🤖 Multi-Agent Architecture

This project uses **CrewAI** to coordinate multiple specialized AI agents working together:

```
User Input (Destination + Budget + Preferences)
        │
        ▼
┌─────────────────────┐
│   Travel Researcher  │  ← searches real-time web for attractions, hotels, weather
│   (Agent 1)          │    using SerperDevTool
└────────┬────────────┘
         │
         ▼
┌─────────────────────┐
│   Itinerary Planner  │  ← organizes findings into a day-by-day travel plan
│   (Agent 2)          │    with budget allocation
└────────┬────────────┘
         │
         ▼
  Final Personalized Itinerary
```

Each agent has a defined **role**, **goal**, and **backstory** — making them behave like specialized travel experts.

## 🚀 How to Run

### Option 1: Google Colab (Recommended)
1. Open `CrewAI_Using_Gemini_Travel_advisor.ipynb` in Google Colab
2. Add your API keys in Colab Secrets:
   - `gemini_key` → your Google Gemini API key
   - `serper_key` → your SerperDev API key
3. Run all cells — the Streamlit app launches via ngrok tunnel

### Option 2: Run Locally
```bash
# Clone the repo
git clone https://github.com/sravyakirthana/ai-travel-planner.git
cd ai-travel-planner

# Install dependencies
pip install crewai crewai-tools streamlit

# Set environment variables
export GOOGLE_API_KEY=your_gemini_key
export SERPER_API_KEY=your_serper_key

# Run the app
streamlit run app.py
```
